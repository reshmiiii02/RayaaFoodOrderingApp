# 🍔 Food Delivery App (Raaya)

## 🚀 Description

Raaya is a food delivery web application that allows users to browse menu items, add products to cart, and simulate an order checkout process with a user-friendly interface.

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

## ✨ Features

* Categorized food menu
* Add to cart functionality
* Dynamic price calculation
* Interactive UI with smooth user experience

## 🎯 Learning Outcome

* Strengthened JavaScript concepts (DOM manipulation)
* Learned how to manage user interactions
* Improved UI/UX design skills
